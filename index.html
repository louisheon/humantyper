FAQ

Questions.

Does this actually fool Google Docs version history?
Yes. HumanTyper uses Chrome's debugger protocol to send real, low-level keyboard input events — the exact same signals as a real person typing. Google Docs records each character as a separate edit, spread over real time. The resulting version history is completely indistinguishable from genuine typing.
What's the 250 character free limit?
The free tier lets you type 250 characters per day — about 2–3 sentences, enough to try it out. The limit resets at midnight. Pro removes it entirely with no daily cap.
Is it a subscription?
No. Pro is a one-time $9 payment. You get a lifetime license key that works on up to 2 devices. No monthly charges, ever.
Why does Chrome show a "DevTools is debugging" banner?
The Chrome Debugger API — which is what makes real keystroke history possible — shows this as a browser safety notice. It disappears the moment typing finishes. It's only visible to you, not in the document or its history.
Does it work on sites other than Google Docs?
Yes. The floating button is Google Docs–only, but the extension toolbar icon works on any website — text inputs, forms, Gmail, Notion, and more.
